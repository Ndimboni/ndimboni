# Hoaxchange Chrome Extension

## Installation Instructions

1.  Open Google Chrome.
2.  Navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked** in the top left corner.
5.  Select the `frontend/extension` directory from this project.
6.  The **Hoaxchange - Scam Detector** extension should now appear in your extensions list.

## Usage

1.  Click the Hoaxchange shield icon in your browser toolbar.
2.  **Login**: Enter your Hoaxchange credentials (you can register on the main website if you don't have an account).
3.  **Check Current Page**: Navigate to any website and click "Scan Current URL" to check if it's safe.
4.  **Manual Check**: Paste any suspicious text, link, or email content into the text area and click "Check Text".
